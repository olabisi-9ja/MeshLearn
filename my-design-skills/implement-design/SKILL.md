# Implement Design Skill

Translates high-fidelity UI design concepts, spatial arrangements, and design specs into high-quality, structured, and modular code blueprints.

## Core Directives

* Always match the layout density and proportions specified in design configurations exactly.
* Parse design system values (such as padding, borders, spacing scales, and border radii) directly into structured, responsive front-end layout blocks.
* Bridge the gap between developer blueprints and pure design assets without losing visual alignment.
