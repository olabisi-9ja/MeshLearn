# Theme Factory Skill

Houses global design token config assets such as CSS variables, color palettes, responsive typography scales, and structural borders.

## Core Directives

* Export variables strictly as cohesive theme tokens. Do not hardcode loose random HEX values.
* Automatically balance contrast ratios across themes (light, dark, elevated) to pass standard web accessibility checks.
* Coordinate layout themes symmetrically to support easy toggle and transition states.
